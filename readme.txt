== Description ==

MT Dev Broken Link Checker Lite is a lightweight plugin that helps you detect and manage broken links on your WordPress site. It checks the links in your posts and pages, ensuring that your visitors always have access to the content they are looking for.

== Installation ==

1. Upload the `mtdev-broken-link-checker-lite` folder to your plugins directory (`/wp-content/plugins/`).
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Go to `Broken Links` in the admin menu to configure the plugin and start scanning for links.

== Frequently Asked Questions ==

= Does this plugin automatically scan links? =

No, currently you will need to manually initiate the scan from the plugin's page in the admin area. An option for automatic scanning is planned for future versions.

= What type of links does it check? =

The plugin checks both internal and external links in your posts and pages.
